Reporting & Examples

Reports consolidate results from scans against approved targets in consent.txt.

Typical workflow:

  • Run port scans (e.g. localhost, example.com), stress tests, and discovery against allowed targets.
  • Leave input_dir as output to include all recent module JSON files.

1) Choose the output directory (usually output/).
2) Select report format (DOCX, PDF, or both).
3) Generate and then open the report files to review consolidated findings.
